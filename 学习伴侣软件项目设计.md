## 一、产品定义

**一句话**：Windows 桌面学习时长统计工具，后台默默记录，打开就是漂亮的统计面板和成就感反馈。

**核心原则**：
- 🚫 不打扰为主 —— 不弹"该学习了"，不搞催促轰炸
- ✅ 精准提醒为辅 —— 只在关键场景温柔提醒（刷娱乐太久 / 晚间空窗），空窗提醒默认关闭
- 🎯 正反馈驱动 —— 进度条、成就、热力图，视觉美感激发成就感
- 🏠 数据完全本地 —— 不上传任何服务器

**数据来源**：ActivityWatch（`localhost:5600`），追踪窗口标题/进程名/浏览器URL。

**运行形态**：关掉窗口 → 缩到系统托盘，后台继续工作，开机自启。

## 三、概览页（Dashboard，默认首页）

页面从上到下依次为：随机鼓励语 → 三科进度条 → 数据卡片 → 科目环形图 → 7天趋势 → 今日时间轴。

### 3.1 三科进度条（页面最上方，核心模块 🔥）

```

│   ████████████░░░░░░░░ 79%  │  │   ████████████████████ 100% │  │   ████████████████████████✨│
│   还差 25m 达标              │  │   已达成！                  │  │   超额 133%！  +50m         │
└─────────────────────────────┘  └─────────────────────────────┘  └─────────────────────────────┘
```

**三种视觉状态**：

| 状态 | 进度条 | 末端 | 下方文字 |
|------|--------|------|---------|
| 🟡 未达标 | 科目色 + 灰色剩余 | 无 | "还差 XXm 达标" |

| ✨ 超额 | 满条 + 金光脉冲动画 | ✨ 金星 | "超额 XXX%！+XXm" |

**双档目标逻辑**：
- 基础目标：用户设定（如物理 2h）
- 挑战目标：基础目标 × 1.5（自动计算，如物理 3h）
- 进度条以基础目标为 100%，超出后金色叠加

### 3.2 超额奖励系统

当某科达到挑战目标时触发：进度条末端金光脉冲动画 2 秒，数值变为金色粗体。

## 四、今日时间轴（概览页底部）

### 4.1 设计理念

- **横向排列**：时间轴从左到右，充分利用电脑宽屏（16:9/16:10），一次看全天
- **信息分层**：默认展示合并后的清晰视图，隐藏噪音；点击可展开看完整原始数据
- **数据不丢失**：合并不等于删除，所有原始数据都在，只是默认不展示

### 4.2 横向布局

```
🕐 今日时间轴                                    [展开所有细节] [仅学习] [全部]

 8:00        10:00       12:00       14:00       16:00       18:00       20:00       22:00
  ├───────────┼───────────┼───────────┼───────────┼───────────┼───────────┼───────────┤
  │                                                       │
  │◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼│░░│◼◼◼◼◼◼◼│░░│◼◼◼◼◼◼◼◼◼◼◼◼◼│░░░░│◼◼◼◼◼◼│  ← 色条=科目色
  │   ⚡ 物理 1.8h │  │  📐 1.2h │  │   ⚡ 物理 2.5h  │    │📖0.8h│
  │                      │          │                  │    │      │
  └─ 展开 ∨ ─────────────┘          └─ 展开 ∨ ────────┘    └──────┘
```

X 轴 = 一天 24 小时时间轴（横向滚动）。Y 轴不堆叠，用不同颜色的色条铺在时间轴上，同一时段多个窗口取持续时间最长的那个显示。

- 每个色条 = 一个合并后的学习段
- 色条颜色 = 科目颜色
- 色条宽度 = 持续时间（按时间刻度比例缩放）
- 色条上方标注：科目图标 + 总时长
- 白色间隙（░░）= 没有学习活动的空白时段

### 4.3 智能合并规则

ActivityWatch 原始数据一天可能有几百条窗口切换记录，必须合并后才能展示。规则如下：

| 规则 | 说明 | 示例 |
|------|------|------|
| **① 同类相邻合并** | 同一科目 + 同一窗口标题 + 时间相邻（间隔 < 2分钟），合并为一段 | 看物理40min → AI问答1min → 继续物理38min → 合并为物理78min，中间AI暂存 |
| **② 噪音过滤** | 持续时间 < 30秒的窗口切换，默认不展示 | 桌面3秒、切窗口1秒 → 丢弃 |
| **③ 间隙保留** | 两段学习之间超过 5 分钟的空档，显示为空白间隙 | 午饭1小时 → 显示为空隙 |
| **④ 穿插暂存** | 学习段中间夹的短暂非学习窗口（< 5分钟，如AI问答、回消息），默认折叠到间隙中，点击展开可见 | 物理→2min微信→物理 → 默认只显示连续的物理条，点击展开才看到中间夹了微信 |

**关键**：规则④中，中间夹的短暂活动**不删除**，只是默认折叠。用户点击色条上的"展开 ∨"按钮后，色条会拆开显示中间的穿插内容。

### 4.4 展开/折叠交互

**默认视图**（合并后）：

```
│◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼◼│  ← 一个连贯的物理色条
│   ⚡ 物理 1.8h        │
│   [展开 ∨]            │
```

**点击"展开"后**：

```
│◼◼◼◼◼◼◼◼◼│💬│◼◼◼◼◼◼◼◼◼│  ← 拆开，中间夹的AI对话露出来
│ 物理 40m  │2m│ 物理 38m  │
│           │   │           │
│  [收起 ∧] │   │           │
```

展开后的 AI 对话（或其他非学习窗口）用细条 + 半透明色显示，不抢视觉焦点，但信息完整保留。

### 4.5 筛选切换

时间轴右上角三个筛选按钮：

| 按钮 | 效果 |
|------|------|
| **仅学习**（默认） | 只显示学习相关条目，过滤娱乐、闲聊 |
| **全部** | 显示所有记录，包括娱乐、未分类 |
| **展开所有细节** | 所有合并段全部展开，显示原始粒度 |

### 4.6 未分类条目标记

来自百度网盘的"视频播放"条目，如果当前没有通过托盘切换科目来指定，会显示为灰色 ❓ 标记。用户可点击灰色条上的标记按钮来指定科目：

```
│◼◼◼◼◼◼◼◼◼◼◼◼◼│  ← 灰色色条
│  ❓ 视频播放 52m  │
│  [标为物理▾]     │  ← 点击弹出下拉选科
```

标记后色条变为对应科目颜色，并且同一天内后续所有"视频播放"自动归为该科目（相当于事后补设了当日默认科目）。

## 八、系统托盘与科目快捷切换（核心交互 🔑）

### 8.1 托盘图标

- 默认图标：绿色叶子 🍃
- 当前设为某科目后：图标颜色变为该科目色（蓝=数学、橙=物理、玫红=英语）
- 有未读成就：角标小红点

### 8.2 右键菜单

```
┌─────────────────────┐
│ 📐 当前科目：数学    │  ← 勾选状态，蓝色高亮
│ ⚡ 切换为物理        │
│ 📖 切换为英语        │
│ ❓ 不指定（手动标记） │
│ ─────────────────── │
│ 打开澜山              │
│ ─────────────────── │
│ 关于                │
│ 退出                │
└─────────────────────┘
```

### 8.3 快捷切换逻辑

**核心规则**：托盘当前科目 > 标题关键词匹配 > 默认未分类

| 场景 | 托盘状态 | 窗口标题 | 分类结果 | 原因 |
|------|---------|---------|---------|------|
| 看数学网课 | 📐 数学 | "视频播放" | 📐 数学 | 托盘覆盖 |
| 看物理网课 | ⚡ 物理 | "视频播放" | ⚡ 物理 | 托盘覆盖 |
| 看B站英语 | 任何状态 | "FREE高考英语——词汇" | 📖 英语 | 标题关键词优先 |
| 看B站数学 | 任何状态 | "小火车数学暑假第5讲" | 📐 数学 | 标题关键词优先 |
| 切换科目时 | 右键 → 点一下 | — | 切换完成 | 一天 2-3 次 |

**关键优先级说明**：标题明确匹配到科目的（如B站视频标题含"物理""数学""FREE高考英语"），**不受托盘当前科目影响**，始终按关键词归类。托盘只覆盖那些标题无法识别的条目（即百度网盘的"视频播放"）。

### 8.4 使用场景

> **早上 8:30**：准备看数学。右键托盘 → 点"📐 切换为数学"。图标变蓝。
> 
> 接下来看网盘里的小火车数学，所有"视频播放"自动归为数学。
>
> **10:30**：数学看完，换物理。右键托盘 → 点"⚡ 切换为物理"。图标变橙。
>
> 继续看网盘夏梦迪物理，"视频播放"自动归为物理。
>
> **下午**：看 B站陶然英语。不用管托盘——B站标题自带"FREE高考英语"，自动识别。
>
> 全天操作：两次右键点击，总共不到 5 秒。

### 8.5 忘记切换的兜底

如果忘了切（比如在看物理但托盘还是数学色），两条补救路径：
1. 图标颜色不对 → 一眼发现 → 右键切回来
2. 已经记录错了 → 打开今天的时间轴，点击灰色色条旁边的标记按钮改回来

### 8.6 窗口行为

- 关闭 = 隐藏到托盘，不退出
- 退出只能通过托盘右键菜单
- 首次关闭时提示："澜山会在后台继续记录，托盘图标可以随时打开 🍃"

## 九、智能提醒（P2）

### 9.1 娱乐超时提醒

连续刷娱乐 ≥ 设定阈值（默认 30 分钟），托盘弹出气泡：

> "已经刷了 30 分钟啦~要回来吗？🍃"

5 秒消失，不拦截屏幕。同一时段（2 小时内）只提醒一次。

### 9.2 晚间空窗提醒

默认关闭。开启后，若晚上无学习记录，21:00 提醒：

> "今晚还没有学习记录呢~不过累了就好好休息 💤"

每周最多 2 次。
